# AI Engineer Portfolio

Modern MERN Stack Portfolio with AI Assistant.

## Features

- Responsive Design
- Dark Mode
- AI Portfolio Assistant
- GitHub Contribution Graph
- Testimonials
- Animated Counters
- Contact Form
- Resume Download

## Setup

### Install Dependencies

```bash
npm install
```

### Frontend

```bash
cd client
npm install
```

### Backend

```bash
cd server
npm install
```

### Run Project

```bash
npm run dev
```

Frontend:

http://localhost:5173

Backend:

http://localhost:5000
